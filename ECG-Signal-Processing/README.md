❤️ ECG Signal Processing (MATLAB)

> 📡 Filtering | 📊 FFT | ❤️ R-Peak Detection | 📉 Noise Removal

📌 Overview

This project focuses on processing ECG (Electrocardiogram) signals to remove noise and accurately detect R-peaks for heart rate estimation.

It demonstrates how signal processing techniques can improve biomedical signal quality and enable reliable analysis.


⚙️ Features

* 📉 Baseline drift removal
* 🎚️ Butterworth filtering
* 📊 Frequency analysis (FFT)
* ❤️ R-peak detection


🧠 Methodology

1. Load ECG signal
2. Remove baseline drift using high-pass filter
3. Remove noise using low-pass Butterworth filter
4. Analyze signal in frequency domain (FFT)
5. Detect R-peaks using peak detection algorithm
